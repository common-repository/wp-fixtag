== Description ==
Automatically add tags to posts . tags can be define by admin/or you can just let it extract tags from post content and add them to post . this plugin do this operations automatically when you publish a new post or edit old posts
Please feel free to offer any feature suggestions you might have and I will consider them for future releases.

== Installation ==
Take the easy route and install through the WordPress plugin adder OR :

1. Download the .zip file and upload the unzipped folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. WP-FixTag should automatically be added to your site and WordPress.
